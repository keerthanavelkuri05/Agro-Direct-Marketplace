# Agro Direct Marketplace

## Project Overview
Agro Direct is a Farmer-to-Consumer Marketplace developed using Python.

## Features
- Farmer Registration
- Product Listing
- View Available Products
- Purchase Products
- Stock Management

## Technology Used
- Python
- Jupyter Notebook

## Benefits
- Direct connection between farmers and consumers
- Better prices for farmers
- Fresh products for consumers
